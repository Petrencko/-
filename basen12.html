Завдання №12

Написать функцию, для корректировки поля "url" в форме. Если поле не начиниается с "http://", следует добавить эту подстроку в начало. Если она там уже есть, ничего делать не нужно. Результат вывести в поле результата.   Далее Вам необходимо создать форму с именем "Test", в которой помещены:   поле ввода с именем "In"; поле ввода с именем "Result"; кнопка "Enter", которая при нажатии должна вызывать Вашу функцию, передав ей в качестве аргумента строку из поля "In", а результат работы функции вывести в поле "Result". Более конкретно, обработка нажатия кнопки должна выглядеть так: onclick="document.getElementById('Result').value=имя_функции(document.getElementById('In') .value)" где имя_функции необходимо заменить на имя Вашей функции.
